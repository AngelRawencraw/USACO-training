[解题报告 ]
动态规划+ 数论,首先这是一个简单的重复背包模型 ,因为我们的确需要知道最大的不可行数字之前的所有情况 ,因此需要 DP,但是 2x10^11的规模太过于恐怖,所以需要数论的知识进行剪枝 ,也就是定界 ,使上界缩小 .第二我们需要一个方法来判断是否拥有无限解 ,先说这个 :如果 ax+by=c(a,b,c,x,y均是>=0 的Z),那么存在 (a,b)=d,并且d|c, 若d!=1,一定有 N>c且d|N 不成立,而且有一个 N不成立 ,就有无穷个 N的倍数也不成立 ,这就是斐蜀定理 ,用这个可以判断有无无限解 .接着是限定上界 :假设存在 vi与 vj互质 ,则 vi*vj之后的一定可以连续得到 ,那么根据这个定理就可以限定上界 ,取得最大的就可以 256*256,而且远远小于USACO 的描述中的= =|||
讨论一下, 不需要数论的 :对于上界 ,我们现在可以知道远远小于 2x10^11,所以像USACO 的第一个题解那样 pos < 2000000000= =|||,是可行的. 需要判断周期 ,因为最大 256,在这个周期内都可行了 ,那么往后也可行了 .而判断是否无限解 ,只要判断不可行数与种类的周期增长是否一样就行 .

#include <fstream>
#define MAX(a,b) ((a)>(b)?(a):(b))
using namespace std;
ofstream fout ("nuggets.out",ios::out);
ifstream fin ("nuggets.in",ios::in);
bool dp[1000001];
long n,v[11],gcd,Max,nonum=0;
long Gcd(long a,long b){
    if(a%b==0) return b;
    else return Gcd(b,a%b);
}
int main()
{
    fin>>n;
    if(n==1) {fout<<0<<endl; return 0;}
    for(int i=1;i<=n;i++) fin>>v[i];
    gcd=Gcd(v[1],v[2]);
    for(int i=3;i<=n;i++) gcd=Gcd(gcd,v[i]);
    if(gcd!=1){fout<<0<<endl; return 0;}
    for(int i=1;i<n;i++) Max=MAX(Max,v[i]);
    dp[0]=true;
    for(int i=1;i<=n;i++)
        for(long j=1;j<=Max*Max;j++)
                                                 if(j-v[i]>=0) dp[j]=dp[j] || dp[j-v[i]];
                                                 else dp[j]=dp[j];
    for(long k=Max*Max;k>=1;k--)
        if(!dp[k]) {nonum=k;break ;}
    fout<<nonum<<endl;
    return 0;
}
