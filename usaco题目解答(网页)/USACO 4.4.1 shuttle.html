USACO 4.4.1 shuttle

[解题报告 ]
这类题目一定要认真分析题目的例子，首先通过观察发现中途一定会有 _WBWBWB... 这样的状态，然后转换到 BWBWBW....BW_ 就会很容易，于是就有了对称性，这意味着如果 BFS,不必完全枚举完成，只要达到 _WB...WB的状态即可，余下的可以直接构造。
这道题目还有更好的构造法，具体请参见 nocow.

[程序清单 ]
#include<fstream>
using namespace std;
ofstream fout ("shuttle.out",ios::out);
ifstream fin ("shuttle.in",ios::in);
int n,rect[500],dis,head,len,rec;
int main()
{
    fin>>n;
    for(int i=1;i<=n+1;i++){
        dis=(i&1?-2:2);
        head=(i&1?n+i:n-i+2);
        for(int j=1;j<=i;j++)
            rect[++len]=head+dis*(j-1);
    }
    for(int i=n;i>=1;i--)
        for(int j=1;j<=i;j++)
            rect[++len]=rect[i*(i-1)/2+j];
    for(int i=2;i<len;i++)
        (++rec%20?fout<<rect[i]<< ' ':fout<<rect[i]<<endl);
    fout<<rect[len]<<endl;
                 return 0;
}
